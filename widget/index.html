IT Director — This is a test page for the Ask Steve AI chat widget.

About

This page simulates embedding the Ask Steve AI widget on stevenpinto.com. Click the blue chat bubble in the bottom-right corner to try it out.

Embed Code

Add this to any webpage to embed the widget:

<script src="https://stevenpinto.com/widget/steve-ai-widget.js"></script>
<steve-ai-widget mode="public" api-url="https://api.stevenpinto.com"></steve-ai-widget>

Configuration

Attributes:

  • mode — "public" (default) or "private"
  • api-url — Backend API URL (default: http://localhost:3001)
  • auth-token — Google auth token (private mode only)
